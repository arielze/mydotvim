##Key bindings
###General
- leader ,
- Ctrl e traverse over open buffers

###QuickFix
- ,H first
- ,J last
- ,h previous
- ,j next

###Window navigation
From astrails/dotvim

- Ctrl h left window
- Ctrl j lower window
- Ctrl l left window
- Ctrl k upper window

###NerdTree
- Ctrl n toggle NerdTree
- ,t reveal in tree

###NERDCommander 
From astrails/dotvim
- ,/ toggle comment

###CtrlP
- ,o open mixed mode
- ,O open tags mode
- Ctrl o reload & mixed search

###AG search
- Ctrl f search word under ursor
